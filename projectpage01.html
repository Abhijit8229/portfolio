Google Drive Clone

Overview

The Google Drive Clone project is an attempt to mimic the core functionalities of Google Drive using React and Firebase. This application allows users to upload files, create folders, and manage their data just like in the real Google Drive.

Key Features

  • User authentication using Firebase Authentication
  • File uploading and real-time storage management
  • Folder structure to organize files
  • Responsive design, ensuring usability across devices
  • Firebase Firestore integration for database storage

Tech Stack

Frontend: React
Backend: Firebase (Firestore, Firebase Storage, Firebase Authentication)
Hosting: Firebase Hosting
