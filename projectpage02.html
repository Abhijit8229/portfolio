Weather App

Overview

This Weather App, built with SvelteKit, allows users to check the weather in any city. The app uses OpenWeatherMap API to fetch and display current weather data, as well as forecasts for the next few days.

Key Features

  • Real-time weather data for any city
  • 7-day weather forecast
  • Responsive design for mobile and desktop
  • Clean and minimalistic UI

Tech Stack

Frontend: SvelteKit
API: OpenWeatherMap API
Hosting: Vercel
